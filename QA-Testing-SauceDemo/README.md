# QA Testing Project – SauceDemo

📌 Overview
This project is a manual QA testing practice for the SauceDemo website.  
It includes test plan, test cases, and bug report to demonstrate basic software testing skills.

📁 Project Structure
- test-plan.md – Contains project scope, objectives, and testing types.
- test-cases.md – Includes structured functional and negative test cases.
- bug-report.md – Documents detected issues with steps to reproduce and expected results.
